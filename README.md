# Russian NER for Personal Names (ФИО)

Извлечение фамилий, имён и отчеств из русского текста с использованием предобученной модели `XLM-RoBERTa`.

## 📋 Описание проекта

Проект решает задачу извлечения персон (ФИО) из текста с помощью NER. 
Выбрал модель `FacebookAI/xlm-roberta-large-finetuned-conll03-english` обрабатывает русскоязычные тексты и возвращает структурированные данные.

### Файлы:
* Основной ресерч и работа: [research.ipynb](research.ipynb)
* Готовый скрипт для запуска: [main.py](main.py)

**Точность по словам** = `77.8%`

## Архитектура решения
`EasyOCR` + `NER`. `EasyOCR` выбрал из-за баланса скорости и качества, `Tesseract` устарел.

Для детекции использовал **beamSearch**, ибо **greedy** хоть и быстрее, качестно ниже, а **wordbeamsearch** уже лишним будет. Предобработка в ЧБ улучшила распознавание.

`NER` модель выбрал для гибкости и устойчивости потому что, сначала хотел через регулярки делать, но это они сломаются при изменении положении документа, а `NER` выделяет ФИО по смыслу. 

Для выбора модели, сначала провер ресерч, то есть сравнил модели на hugging face, по балансу количеству параметров и легкости. 

После сравнения нескольких моделей выбрал `FacebookAI/xlm-roberta-large-finetuned-conll03-english` (600M параметров) — она лучше справилась с редкими символами и длинными отчествами. 

Изначально выбрал `Gherman/bert-base-NER-Russian`, ибо она легче, но протестировав, понял, что без улучшения никак, а вот фесбуковская модель работает гораздо лучше прямо из коробки!
Ну в итоге ее и выбрал, ибо нужно было лучшее распознавание!

## Как улучшить?
1. Тут от специфики задачи, если нам нужен инференс **быстрее** то возьмем от `Gherman`, а если позволительно взять модель лучше, то от фейсбук!
2. Можно докрутить пре **процессинг**, то есть еще лучше очищать картинку до передачи ее в `easyosr`, масштабировать например, исправить наклон, убрать тени и засветы и т.д.
3. **Дообучить** модель на русском датасете с именами, что по идее должно научить модель корректно разпознавать редкие символы в именах (типо: ь, ъ и т.д) и избавит от обрезания длинных ФИО.
4. **Собрать больше данных** под конкретные СТС.
5. Ну или просто **взять модель пожирнее** и дообучить ее, но это кажется лишним, ибо второго шага уже достаточно!

## 🚀 Установка и запуск

### 1. Установка Poetry

```bash
# Установка Poetry (если ещё не установлен)
curl -sSL https://install.python-poetry.org | python3 -

# Или через pip
pip install poetry
```
**Документация:** [`https://python-poetry.org/docs/`](https://python-poetry.org/docs/)


### 2. Клонирование репозитория
``` bash
git clone https://github.com/your-username/ner-russian-names.git
cd ner-russian-names
```
### 3.Установка всех зависимостей через Poetry
```bash
poetry install
```

### 4. Активация виртуального окружения
```bash
eval $(poetry env activate)

python main.py
```
